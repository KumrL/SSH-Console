<h1>SSH-Console</h1>
<p>Description: Graphical SSH client designed for remote administration, improving the user experience by visually separating command input, standard output, and errors into independent panels. It features command saving and the ability to export error and full logs to .txt files for greater clarity and control.</p>
<p>Descripción: Cliente SSH con interfaz gráfica diseñado para la administración remota, mejorando la experiencia del usuario al separar visualmente la entrada de comandos, la salida estándar y los errores en paneles independientes. Cuenta con guardado de comandos y la capacidad de exportar errores y registros completos a archivos .txt para mayor claridad y control.</p>
